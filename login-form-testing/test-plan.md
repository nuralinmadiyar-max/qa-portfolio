# Тест-план: Форма логина (saucedemo.com)

**1. Объект тестирования:** Форма логина на saucedemo.com (поля username, password, кнопка Login)

**2. Цели тестирования:** Проверить корректность работы формы логина при вводе валидных и невалидных данных (пустые поля, неверные учётные данные, спецсимволы, заблокированные пользователи и т.д.)

**3. Scope:**
- Входит: тестирование полей username, password и кнопки Login
- Не входит: тестирование ссылки "Forgot Password", тестирование страниц после успешного входа

**4. Виды тестирования:** Функциональное, позитивное, негативное, тестирование граничных значений

**5. Окружение:** Google Chrome, Windows 11

**6. Критерии начала/окончания:**
- Начало: сайт доступен, форма отображается корректно
- Окончание: все 11 тест-кейсов выполнены с зафиксированным статусом, баг-репорты оформлены при обнаружении расхождений

**7. Риски:** Технические неполадки, изменение поведения сайта без предупреждения
